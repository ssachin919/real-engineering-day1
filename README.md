# Real Engineering Day 1 Demo

This is a simple course interest landing page built to demonstrate the real software development workflow.

## Workflow covered

Idea → Code → GitHub → Review → Deploy → Bug Fix

## Roadmap:

- Week 1
    We will learn Cursor IDE, terminal basics, Git commits, GitHub repos, branches, pull requests, and survival Git commands.

- Week 2: frontend
    We will learn HTML, CSS, JavaScript, React, state, lists, Next.js, static rendering, and deployment on Vercel.

- Week 3: backend
    We will learn APIs, FastAPI, Flask, Django, NestJS, and how frontend talks to backend.

- Week 4: real prod systems
    We will learn databases, PostgreSQL, ORM, authentication, authorization, backend deployment, serverless functions, CI/CD, and finally a complete feature-to-production workflow.
